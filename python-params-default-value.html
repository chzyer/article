关于Python的参数默认值

November 07, 2012

最近相继遇到几个python的坑...贴一段有意思的代码

def a(b=[]):
    b.append(1)
    return b
print a()
print a()

输出是什么?

>> [1]
>> [1, 1]

我对python没系统的了解过, 大概只能猜到函数在调用前应该已经生成了这个对象, 而且直接返回指针, 如果不知道的话很容易撞坑上

def a(b=None):
    if not b:
        b = []
    b.append(1)
    return b

a正常应该这样写, 不过这个特性或许可以利用.. 比如, 好吧没想到. 不过传入一个初始化的对象的话不会有这个问题。。
